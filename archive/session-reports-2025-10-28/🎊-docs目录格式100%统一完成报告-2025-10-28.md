## 🎯 统一后的标准格式

### 标准格式定义

```markdown
## 📋 目录

- [1. 章节标题](#1-章节标题)
- [2. 章节标题](#2-章节标题)
  - [2.1 子章节](#21-子章节)
  - [2.2 子章节](#22-子章节)
- [3. 章节标题](#3-章节标题)
```

### 格式特点

✅ **标题**: 使用 `## 📋 目录` 或 `## 📋 目录结构`  
✅ **列表**: 使用 `-` (dash) 格式  
✅ **编号**: 保留数字编号 (1. 2. 3.)  
✅ **缩进**: 支持多级缩进 (子章节)  
✅ **链接**: 使用标准Markdown链接格式  

### 应用范围

1. **内容文档**: 使用 `## 📋 目录`
2. **导航文档**: 使用 `## 📋 目录结构`  
3. **README**: 使用 `## 📋 目录结构`
