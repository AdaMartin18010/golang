## 8.1.1 📚 **理论分析**

- 基准测试（Benchmark）用于度量函数性能，找出瓶颈。
- Go内置testing.B支持基准测试，配合pprof可深入分析。

## 8.1.2 🛠️ **常用命令**

- 运行所有基准测试：

  ```bash
  go test -bench .
  ```

- 采集CPU/内存profile：

  ```bash
  go test -bench . -cpuprofile cpu.out -memprofile mem.out
  ```

## 8.1.3 💻 **代码示例**

### 8.1.3.1 **基准测试模板**

```go
func BenchmarkAdd(b *testing.B) {
    for i := 0; i < b.N; i++ {
        Add(1, 2)
    }
}
```

### 8.1.3.2 **并发基准测试**

```go
func BenchmarkParallelAdd(b *testing.B) {
    b.RunParallel(func(pb *testing.PB) {
        for pb.Next() {
            Add(1, 2)
        }
    })
}
```

## 8.1.4 🎯 **最佳实践**

- 基准测试环境应稳定，避免外部干扰
- 多次运行取平均，关注方差
- 配合pprof分析热点

## 8.1.5 🔍 **常见问题**

- Q: b.N是什么？
  A: Go自动调整循环次数，保证测试精度
- Q: 如何分析基准测试结果？
  A: 结合pprof、top、list等命令
