# Wire 依赖注入示例

> **版本**: v1.0
> **日期**: 2025-01-XX

---

## 📋 概述

本示例展示如何使用 Google Wire 进行依赖注入，组织框架的各种能力。

---

## 🎯 功能展示

本示例展示了：

1. ✅ **Wire 依赖注入** - 使用 Wire 管理依赖关系
2. ✅ **Provider 函数** - 定义各种 Provider 函数
3. ✅ **依赖关系** - 展示依赖关系的组织方式

---

## 🚀 使用步骤

### 1. 生成 Wire 代码

```bash
cd examples/framework-usage/wire-example
go generate
```

### 2. 运行示例

```bash
go run .
```
