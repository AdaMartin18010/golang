## 1. 数据库类型对比

### MySQL vs PostgreSQL vs MongoDB

| 维度 | MySQL | PostgreSQL | MongoDB |
|------|-------|-----------|---------|
| **类型** | 关系型 | 关系型 | 文档型 |
| **性能** | 高 | 高 | 极高 |
| **ACID** | 强 | 强 | 弱 |
| **扩展性** | 中 | 强 | 极强 |
| **适用场景** | Web应用 | 复杂查询 | 大数据 |

## 2. ORM框架对比
