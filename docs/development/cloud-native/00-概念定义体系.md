## 2. 概念定义

### 云原生12要素

**定义**: 构建SaaS应用的方法论

**核心要素**:

1. 代码库 (Codebase)
2. 依赖管理 (Dependencies)
3. 配置 (Config)
4. 后端服务 (Backing services)
5. 构建/发布/运行 (Build, release, run)
...

---

### GitOps

**定义**: 使用Git作为单一事实来源的运维模式

**流程**:

```text
Git提交 → CI构建 → 更新配置仓库 → ArgoCD同步 → K8s部署
```

---

### Sidecar模式

**定义**: 在Pod中运行辅助容器

**用途**: 日志收集、代理、监控
