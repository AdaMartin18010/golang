## 📝 下一步操作

### 1. 生成 gRPC 代码（必需）

由于 gRPC Handler 使用了 Proto 生成的代码，需要先运行代码生成：

```bash
# 安装 Protocol Buffers 编译器（如果未安装）
# macOS
brew install protobuf

# Linux
sudo apt-get install protobuf-compiler

# 安装 Go 插件
go install google.golang.org/protobuf/cmd/protoc-gen-go@latest
go install google.golang.org/grpc/cmd/protoc-gen-go-grpc@latest

# 生成 gRPC 代码
make generate-grpc
```

### 2. 取消注释 gRPC 服务注册

生成 Proto 代码后，编辑 `cmd/grpc-server/main.go`，取消注释以下代码：

```go
// 取消注释这些行
userHandler := handlers.NewUserHandler(userService)
userpb.RegisterUserServiceServer(grpcServer, userHandler)

healthHandler := handlers.NewHealthHandler()
healthpb.RegisterHealthServiceServer(grpcServer, healthHandler)
```

### 3. 测试验证

```bash
# 启动 NATS 服务器（用于测试）
docker run -p 4222:4222 nats:latest

# 运行 NATS 测试
go test ./internal/infrastructure/messaging/nats/...

# 运行 gRPC 服务器
go run ./cmd/grpc-server
```
