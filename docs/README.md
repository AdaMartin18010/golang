## 🚀 快速开始

### 3分钟上手

```go
// 1. 安装Go (推荐Go 1.21+)
go version

// 2. 创建第一个程序
mkdir hello
cd hello
go mod init hello

// 3. 编写代码 (main.go)
package main

import "fmt"

func main() {
    fmt.Println("Hello, Go!")
}

// 4. 运行程序
go run main.go
```
