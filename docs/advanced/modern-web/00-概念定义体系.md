## 2. 概念定义

### WebSocket

**定义**: 全双工实时通信协议

**示例**:
```go
import "github.com/gorilla/websocket"

upgrader := websocket.Upgrader{}
conn, _ := upgrader.Upgrade(w, r, nil)

// 读取消息
_, message, _ := conn.ReadMessage()

// 发送消息
conn.WriteMessage(websocket.TextMessage, []byte("Hello"))
```

---

### GraphQL

**定义**: API查询语言

**示例**:
```graphql
query {
  user(id: 1) {
    name
    email
    posts {
      title
    }
  }
}
```

---

### API网关

**定义**: 统一API入口

**功能**:
- 路由
- 认证/授权
- 限流
- 熔断
