## 🚀 快速示例

```go
// 链表
type Node struct {
    Val  int
    Next *Node
}

// 二叉树
type TreeNode struct {
    Val   int
    Left  *TreeNode
    Right *TreeNode
}

// 图
type Graph map[int][]int
```
