```text

│                     语法层 (Syntax Layer)                        │
├──────────────┬──────────────────┬───────────────────────────────┤

│ · 常量定义   │ · 接口           │ · channel                      │
│ · 函数定义   │ · 方法           │ · select                       │

│ · 导入       │ · 组合           │ · context                      │
└──────────────┴──────────────────┴───────────────────────────────┘
                        ↓
┌────────────────────────────────────────────────────────────────┐
│                    语义层 (Semantic Layer)                       │
├──────────────┬──────────────────┬───────────────────────────────┤
│  操作语义     │   指称语义        │      公理语义                  │
├──────────────┼──────────────────┼───────────────────────────────┤
│ · 小步语义   │ · 领域理论       │ · Hoare逻辑                    │

│              │                  │                                │

│   e ⇓ v      │   λx.e          │   Inv(L)                       │
│              │                  │                                │
│ · 求值策略   │ · 连续语义       │ · 前后条件                     │
│   严格/惰性  │   递归函数       │   断言推理                     │
└──────────────┴──────────────────┴───────────────────────────────┘
                        ↓
┌────────────────────────────────────────────────────────────────┐
│                 形式化层 (Formalization Layer)                   │
├──────────────┬──────────────────┬───────────────────────────────┤
│  类型安全     │   并发正确性      │     编译器优化                 │
├──────────────┼──────────────────┼───────────────────────────────┤
│ Progress:    │ CSP验证:         │ 语义保持:                      │
│ ⊢ e:T →     │ P || Q           │ [[P]] ≡ [[P']]                │
│ value(e) ∨   │ deadlock-free    │                                │
│ ∃e'. e→e'   │                  │ 性能提升:                      │
│              │ 数据竞争:         │ time(P') < time(P)            │
│ Preservation:│ ¬(op1 ∥ op2)    │                                │
│ ⊢ e:T ∧     │                  │ 正确性:                        │
│ e→e' →      │ 死锁自由:         │ ∀input. P(input) =            │
│ ⊢ e':T      │ ∄circular wait   │         P'(input)              │
└──────────────┴──────────────────┴───────────────────────────────┘
```
