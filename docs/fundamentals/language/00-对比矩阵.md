| 特性 | Go | Python | Java | C++ |

| **编译速度** | ⭐⭐⭐⭐⭐ | N/A | ⭐⭐ | ⭐ |

| **学习曲线** | ⭐⭐ | ⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
