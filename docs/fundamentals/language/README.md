## 🚀 快速开始

### Hello World

```go
package main
import "fmt"
func main() {
    fmt.Println("Hello, Go 1.25.3!")
}
```

### 并发示例

```go
package main

import (
    "fmt"
    "time"
)

func main() {
    ch := make(chan string)

    go func() {
        time.Sleep(1 * time.Second)
        ch <- "Hello from Goroutine!"
    }()

    msg := <-ch
    fmt.Println(msg)
}
```
