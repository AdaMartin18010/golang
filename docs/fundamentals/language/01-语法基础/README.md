## 🚀 快速示例

### 变量声明

```go
var x int = 10
y := 20
const Pi = 3.14
```

### 流程控制

```go
if x > 10 {
    fmt.Println("x > 10")
}

for i := 0; i < 10; i++ {
    fmt.Println(i)
}
```

### 函数

```go
func add(a, b int) int {
    return a + b
}

func divide(a, b int) (int, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}
```
