## 二、概念依赖关系图

```
Go 1.26 学习依赖图
────────────────────────────────────────

基础层 (必须先掌握)
├── Go基础语法
│   └── 变量、类型、控制流
├── 指针基础
│   └── &T 和 *T 的区别
└── 泛型基础 (Go 1.18+)
    └── type T[E any]

进阶层 (Go 1.26新特性)
├── new(expr) [依赖: 指针基础]
│   ├── 语法: new(Expression)
│   ├── 语义: &tmp where tmp = E
│   └── 应用: 可选字段初始化
│
└── 递归泛型约束 [依赖: 泛型基础]
    ├── 理论: 最小不动点
    ├── 语法: type C[T C[T]]
    └── 应用: 树/图通用算法

实践层 (应用场景)
├── Web API开发
│   └── new(expr)用于可选字段
├── 数据结构库
│   └── 递归泛型用于通用实现
├── 加密通信
│   └── HPKE用于安全传输
└── 代码现代化
    └── go fix用于项目升级
```
