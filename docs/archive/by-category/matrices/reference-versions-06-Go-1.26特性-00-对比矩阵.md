## 新语法对比

| 场景 | Go 1.25 | Go 1.26 | 优势 |
|------|---------|---------|------|
| 创建指针 | `&T{...}` | `new(T{...})` | 选择更多 |
| 可选字段 | `tmp := f(); p.Field = &tmp` | `p.Field = new(f())` | 更简洁 |
| 树遍历 | `interface{}` + 类型断言 | `TreeNode[T TreeNode[T]]` | 类型安全 |
| 错误断言 | `errors.As(err, &myErr)` | `errors.AsType[*MyErr](err)` | 更简洁 |
