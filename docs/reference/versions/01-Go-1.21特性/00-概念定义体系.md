## 核心概念

### 1. PGO (Profile-Guided Optimization)

**定义**: 基于实际运行profile的编译优化

**工作流程**:
```
1. 收集profile → 2. 编译优化 → 3. 性能提升
```

**实现**:
```bash
# 收集
go test -cpuprofile=default.pgo

# 编译（自动检测default.pgo）
go build -pgo=default.pgo
```

---

### 2. min/max内置函数

**定义**: 泛型最值计算

**类型约束**:
```go
type Ordered interface {
    ~int | ~float64 | ~string | ...
}
```

---

### 3. clear内置函数

**定义**: 清空map或slice

**语义**:
```go
// map: 删除所有键值对
clear(m)  // len(m) == 0

// slice: 零值化所有元素
clear(s)  // 元素变为零值
```
