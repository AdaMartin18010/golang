## Go 1.25 vs Go 1.24

| 维度 | Go 1.24 | Go 1.25.3 | 提升 |
|------|---------|-----------|------|
| **GOMAXPROCS** | 静态 | 容器感知 | ⭐⭐⭐⭐⭐ |
| **GC** | 标准 | +greenteagc | ⭐⭐⭐⭐⭐ |
| **Nil检查** | 基础 | 改进 | ⭐⭐⭐⭐ |
| **调试信息** | DWARFv4 | DWARFv5 | ⭐⭐⭐⭐ |
| **语言规范** | 完整 | 简化 | ⭐⭐⭐ |
