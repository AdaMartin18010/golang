## 迭代器对比

### 传统 vs 迭代器

| 方式 | 灵活性 | 易用性 | 性能 |
|------|-------|-------|------|
| **传统切片** | ⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| **自定义迭代** | ⭐⭐⭐ | ⭐⭐ | ⭐⭐⭐ |
| **Go 1.23迭代器** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ |

**示例对比**:
```go
// Go 1.22 - 手动实现
type Iterator struct {
    data []int
    pos  int
}

// Go 1.23 - 使用迭代器
for v := range slices.Backward(data) {
    // 简洁！
}
```
