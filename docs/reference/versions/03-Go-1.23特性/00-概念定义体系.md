## 核心概念

### 1. 迭代器 (Iterators)

**定义**: 通过函数表示的可迭代序列

**类型**:

```go
// 单值迭代器
type Seq[V any] func(yield func(V) bool)

// 键值对迭代器
type Seq2[K, V any] func(yield func(K, V) bool)
```

**工作原理**:

```text
1. 迭代器函数接收yield
2. 调用yield传递值
3. yield返回false表示停止
4. 支持惰性求值
```

---

### 2. yield函数

**定义**: 控制迭代流程的回调

**返回值**:

- `true`: 继续迭代
- `false`: 停止迭代

**示例**:

```go
func Counter(n int) iter.Seq[int] {
    return func(yield func(int) bool) {
        for i := 0; i < n; i++ {
            if !yield(i) {  // 检查是否继续
                return
            }
        }
    }
}
```

---

### 3. range over func

**定义**: range可以直接迭代迭代器函数

**语法**:

```go
for v := range iteratorFunc {
    // v是迭代器yield的值
}
```
