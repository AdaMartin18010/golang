## 4. 学习导航

### 按学习路径

**初学者** (从最新版本开始):
```text
1. Go 1.25概览 → 了解当前最新特性
2. Go 1.22 → 学习for循环修复、HTTP路由
3. Go 1.21 → 掌握min/max、slog
4. Go 1.23 → 理解迭代器
```

**从旧版本升级**:
```text
从1.19/1.20:
├─ 必看: Go 1.21 (PGO, min/max, slog)
├─ 必看: Go 1.22 (for循环, HTTP路由)
└─ 推荐: Go 1.23-1.25

从1.21:
├─ 必看: Go 1.22 (for循环修复)
└─ 推荐: Go 1.23-1.25

从1.22/1.23:
└─ 可选: Go 1.24-1.25 (性能优化)
```

**按场景**:
```text
Web开发:
├─ Go 1.22 (HTTP路由增强)
├─ Go 1.21 (log/slog)
└─ Go 1.25 (最新稳定版)

性能优化:
├─ Go 1.21 (PGO)
├─ Go 1.24 (GC优化)
└─ Go 1.25 (持续优化)

并发编程:
├─ Go 1.22 (for循环修复)
└─ Go 1.23 (迭代器)
```
