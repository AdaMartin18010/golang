## 核心概念

### 1. for循环变量per-iteration

**定义**: 每次迭代创建独立的循环变量

**语义变化**:

```go
// Go 1.21: 共享
var v *int
for i := range 3 {
    v = &i  // 最后v指向同一个i
}

// Go 1.22: 独立
var v *int
for i := range 3 {
    v = &i  // 每次i都是新变量
}
```

---

### 2. 整数range

**定义**: `for i := range n` 迭代0到n-1

**语法**:

```go
for i := range n {
    // i: 0, 1, ..., n-1
}
```

---

### 3. HTTP路由增强

**定义**: net/http内置路由匹配

**特性**:

- 通配符: `{param}`
- HTTP方法: `GET /path`
- 参数提取: `r.PathValue("param")`
