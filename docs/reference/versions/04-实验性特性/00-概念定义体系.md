## 核心概念

### GOEXPERIMENT

**形式化定义**:
```go
type Experiment struct {
    Name        string
    Description string
    Status      ExperimentStatus
    Since       string  // Go版本
}

type ExperimentStatus int
const (
    Proposed ExperimentStatus = iota  // 提案
    InProgress                         // 进行中
    Enabled                            // 可用
    Graduated                          // 毕业(正式版)
    Deprecated                         // 废弃
)
```

**使用方式**:
```bash
# 启用单个实验
GOEXPERIMENT=newinliner go build

# 启用多个实验
GOEXPERIMENT=exp1,exp2 go build

# 禁用实验
GOEXPERIMENT=noexp1 go build
```

---

### 实验性特性生命周期

**阶段**:
1. **提案**: GitHub issue讨论
2. **设计**: 详细设计文档
3. **实现**: GOEXPERIMENT实现
4. **反馈**: 社区测试反馈
5. **正式化**: 进入正式版本
6. **废弃**: 不合适则废弃

**时间线**:
- 提案→实现: 1-6个月
- 实验期: 1-2个版本周期
- 总计: 6-18个月
