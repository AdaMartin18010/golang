# Go 1.26 术语表 (Glossary)

> **文档层级**: A-附录 (Appendix)
> **文档类型**: 术语表 (Glossary)
> **最后更新**: 2026-03-06

---

## A

**Axiom (公理)** - 不证自明的基础命题，作为推理的起点。

## C

**CGO** - Go与C代码互操作的机制。

**Concurrent Marking (并发标记)** - 与用户程序并发执行的GC标记阶段。

**Constraint (约束)** - 对泛型类型参数的限制条件。

## E

**Escape Analysis (逃逸分析)** - 编译时分析确定变量是否在函数外部被引用。

## G

**Generic (泛型)** - 支持类型参数化的编程特性。

**GreenTeaGC** - Go 1.26的并发垃圾回收器。

## H

**HPKE (Hybrid Public Key Encryption)** - 混合公钥加密，RFC 9180标准。

## K

**KEM (Key Encapsulation Mechanism)** - 密钥封装机制。

**KDF (Key Derivation Function)** - 密钥派生函数。

## N

**new(expr)** - Go 1.26扩展的new表达式，支持表达式初始化。

## R

**Recursive Constraint (递归约束)** - 约束定义中引用自身的约束。

## S

**SIMD (Single Instruction Multiple Data)** - 单指令多数据并行处理。

**STW (Stop-The-World)** - GC期间暂停所有goroutine的阶段。

## T

**Theorem (定理)** - 由公理推导出的命题。

**Type Parameter (类型参数)** - 泛型中的参数化类型。

## W

**Write Barrier (写屏障)** - GC期间记录指针变更的机制。
