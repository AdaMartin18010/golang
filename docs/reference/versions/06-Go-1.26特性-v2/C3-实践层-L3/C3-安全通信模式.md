## 一、模式概述

### 1.1 设计目标

使用Go 1.26的`crypto/hpke`构建安全的通信系统，实现：

- 前向保密
- 可选的身份认证
- 高效的混合加密

### 1.2 威胁模型

| 威胁 | 防护能力 |
|------|----------|
| 被动窃听 | ✅ 加密保护 |
| 主动攻击 | ✅ 认证加密 |
| 长期密钥泄露 | ✅ 前向保密 |
| 中间人攻击 | ⚠️ 需要身份认证 |

## 二、基础实现

### 2.1 密钥封装

```go
package secure

import (
    "crypto/hpke"
    "io"
)

// Encapsulator 密钥封装器
type Encapsulator struct {
    kemID    uint16
    kdfID    uint16
    aeadID   uint16
    publicKey []byte
}

// NewEncapsulator 创建封装器
func NewEncapsulator(publicKey []byte) *Encapsulator {
    return &Encapsulator{
        kemID:     hpke.DHKEM_X25519_HKDF_SHA256,
        kdfID:     hpke.HKDF_SHA256,
        aeadID:    hpke.AES_256_GCM,
        publicKey: publicKey,
    }
}

// Encapsulate 封装消息
func (e *Encapsulator) Encapsulate(plaintext, info []byte) ([]byte, []byte, error) {
    return hpke.SingleShotSeal(
        e.kemID,
        e.kdfID,
        e.aeadID,
        e.publicKey,
        plaintext,
        info,
    )
}
```

### 2.2 消息加密服务

```go
// MessageService 加密消息服务
type MessageService struct {
    privKey []byte
    pubKey  []byte
}

// NewMessageService 创建服务
func NewMessageService() (*MessageService, error) {
    pubKey, privKey, err := hpke.GenerateKeyPair(hpke.DHKEM_X25519_HKDF_SHA256)
    if err != nil {
        return nil, err
    }

    return &MessageService{
        privKey: privKey,
        pubKey:  pubKey,
    }, nil
}

// Encrypt 加密消息
func (s *MessageService) Encrypt(
    recipientPubKey,
    plaintext,
    context []byte,
) ([]byte, []byte, error) {
    return hpke.SingleShotSeal(
        hpke.DHKEM_X25519_HKDF_SHA256,
        hpke.HKDF_SHA256,
        hpke.AES_256_GCM,
        recipientPubKey,
        plaintext,
        context,
    )
}

// Decrypt 解密消息
func (s *MessageService) Decrypt(
    encapsulatedKey,
    ciphertext,
    context []byte,
) ([]byte, error) {
    return hpke.SingleShotOpen(
        hpke.DHKEM_X25519_HKDF_SHA256,
        hpke.HKDF_SHA256,
        hpke.AES_256_GCM,
        s.privKey,
        encapsulatedKey,
        ciphertext,
        context,
    )
}
```

## 三、进阶模式

### 3.1 双向认证通信

```go
// AuthMessageService 带认证的通信服务
type AuthMessageService struct {
    privKey      []byte
    pubKey       []byte
    remotePubKey []byte  // 预共享的远程公钥
}

// EncryptAuth 发送认证加密消息
func (s *AuthMessageService) EncryptAuth(plaintext []byte) ([]byte, []byte, error) {
    // 使用Auth模式（发送方认证）
    // 需要双方密钥对
    return hpke.SingleShotAuthSeal(
        hpke.DHKEM_X25519_HKDF_SHA256,
        hpke.HKDF_SHA256,
        hpke.AES_256_GCM,
        s.remotePubKey,
        s.privKey,
        plaintext,
        nil,
    )
}
```

### 3.2 密钥轮换

```go
// RotatingKeyService 支持密钥轮换的服务
type RotatingKeyService struct {
    currentKey   *MessageService
    previousKey  *MessageService
    rotationTime time.Duration
    lastRotation time.Time
}

// RotateKeys 轮换密钥
func (s *RotatingKeyService) RotateKeys() error {
    newService, err := NewMessageService()
    if err != nil {
        return err
    }

    s.previousKey = s.currentKey
    s.currentKey = newService
    s.lastRotation = time.Now()

    return nil
}

// Encrypt 使用当前密钥加密
func (s *RotatingKeyService) Encrypt(recipientPubKey, plaintext []byte) ([]byte, []byte, error) {
    if time.Since(s.lastRotation) > s.rotationTime {
        s.RotateKeys()
    }
    return s.currentKey.Encrypt(recipientPubKey, plaintext, nil)
}
```

## 四、安全最佳实践

### 4.1 密钥管理

```go
// ✅ 安全生成密钥
func generateSecureKey() ([]byte, []byte, error) {
    return hpke.GenerateKeyPair(hpke.DHKEM_X25519_HKDF_SHA256)
}

// ✅ 安全存储（使用密钥管理系统）
func storeKeySecurely(key []byte, kmsClient KMSClient) error {
    encrypted, err := kmsClient.Encrypt(key)
    if err != nil {
        return err
    }
    return storeToSecureStorage(encrypted)
}

// ✅ 安全清除
func clearKey(key []byte) {
    subtle.ZeroBytes(key)
}
```

### 4.2 上下文信息

```go
// ✅ 使用上下文绑定消息
func encryptWithBinding(
    pubKey,
    plaintext []byte,
    senderID,
    timestamp string,
) ([]byte, []byte, error) {
    // 上下文包含发送者和时间戳，防止重放攻击
    info := fmt.Sprintf("sender=%s;time=%s", senderID, timestamp)
    return hpke.SingleShotSeal(
        hpke.DHKEM_X25519_HKDF_SHA256,
        hpke.HKDF_SHA256,
        hpke.AES_256_GCM,
        pubKey,
        plaintext,
        []byte(info),
    )
}
```

## 五、完整示例

### 5.1 加密消息队列

```go
type EncryptedQueue struct {
    service  *MessageService
    backend  QueueBackend
}

func (q *EncryptedQueue) Publish(message []byte) error {
    // 获取接收者公钥
    recipientPubKey := q.backend.GetRecipientKey()

    // 加密消息
    encKey, ciphertext, err := q.service.Encrypt(recipientPubKey, message, nil)
    if err != nil {
        return err
    }

    // 包装消息
    envelope := &EncryptedMessage{
        EncapsulatedKey: encKey,
        Ciphertext:      ciphertext,
        Timestamp:       time.Now().Unix(),
    }

    // 发布
    return q.backend.Publish(serialize(envelope))
}

func (q *EncryptedQueue) Subscribe(handler func([]byte)) error {
    return q.backend.Subscribe(func(data []byte) {
        envelope := deserialize(data)

        // 解密
        plaintext, err := q.service.Decrypt(
            envelope.EncapsulatedKey,
            envelope.Ciphertext,
            nil,
        )
        if err != nil {
            log.Printf("Decryption failed: %v", err)
            return
        }

        handler(plaintext)
    })
}
```
