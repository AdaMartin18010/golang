## 9. 类图（类型系统）

```mermaid
classDiagram
    class Ordered~T~ {
        <<interface>>
        +Less(other T) bool
        +Equal(other T) bool
    }

    class TreeNode~T~ {
        <<interface>>
        +Value() int
        +Children() []T
    }

    class GraphNode~N~ {
        <<interface>>
        +Neighbors() []N
        +EdgeCost(to N) int
    }

    class BinaryNode {
        -value int
        -left *BinaryNode
        -right *BinaryNode
        +Value() int
        +Children() []*BinaryNode
    }

    class IntNode {
        -value int
        +Less(other IntNode) bool
        +Equal(other IntNode) bool
    }

    Ordered~T~ <|-- IntNode
    TreeNode~T~ <|-- BinaryNode
```
