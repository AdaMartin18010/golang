## 实践练习

### 练习1: new(expr) 改写（15分钟）

**任务**: 将以下代码改写为使用 new(expr)

```go
// 原始代码
type Config struct {
    Host *string
    Port *int
}

func createConfig() *Config {
    host := "localhost"
    port := 8080
    return &Config{
        Host: &host,
        Port: &port,
    }
}

// 你的答案:
```

**参考答案**:

```go
func createConfig() *Config {
    return &Config{
        Host: new("localhost"),
        Port: new(8080),
    }
}
```

### 练习2: 递归泛型实现（30分钟）

**任务**: 实现一个通用的树遍历函数

```go
// 要求: 实现 Traverse 函数，支持任意树结构

type Node[T Node[T]] interface {
    Value() string
    Children() []T
}

// 实现这个函数
func Traverse[T Node[T]](root T, visit func(string)) {
    // 你的代码
}

// 测试用 BinaryNode
type BinaryNode struct {
    Val   string
    Left  *BinaryNode
    Right *BinaryNode
}

func (b *BinaryNode) Value() string { return b.Val }
func (b *BinaryNode) Children() []*BinaryNode {
    var children []*BinaryNode
    if b.Left != nil { children = append(children, b.Left) }
    if b.Right != nil { children = append(children, b.Right) }
    return children
}
```

**参考答案**:

```go
func Traverse[T Node[T]](root T, visit func(string)) {
    if root == nil {
        return
    }
    visit(root.Value())
    for _, child := range root.Children() {
        Traverse(child, visit)
    }
}
```

### 练习3: go fix 实践（15分钟）

**任务**: 运行 go fix 现代化代码

```bash
# 1. 创建测试项目
mkdir gofix-demo && cd gofix-demo
go mod init demo

# 2. 创建包含旧代码的文件
cat > main.go << 'EOF'
package main

import (
    "fmt"
    "sort"
)

func main() {
    nums := []int{3, 1, 4, 1, 5}

    // 使用旧方式排序
    sort.Slice(nums, func(i, j int) bool {
        return nums[i] < nums[j]
    })

    // 使用旧方式查找
    found := false
    for _, v := range nums {
        if v == 4 {
            found = true
            break
        }
    }

    fmt.Println(nums, found)
}
EOF

# 3. 运行 go fix
go fix -n .  # 预览
go fix .     # 应用

# 4. 查看结果
cat main.go
```

## 培训考核

### 理论题

1. **new(expr)** 和 **&T{}** 的主要区别是什么？
   - A) 性能差异
   - B) 语法简洁性
   - C) 功能不同
   - D) 内存分配位置

   **答案**: B

2. 递归泛型的主要应用场景是？
   - A) Web 开发
   - B) 数据结构库
   - C) 网络编程
   - D) 文件处理

   **答案**: B

3. Green Tea GC 默认启用后，预期的延迟改善是多少？
   - A) 5-10%
   - B) 10-40%
   - C) 50-60%
   - D) 70-80%

   **答案**: B

### 实践题

1. 使用 new(expr) 改写以下代码：

   ```go
   timeout := 30 * time.Second
   config := &Config{Timeout: &timeout}
   ```

   **答案**: `config := &Config{Timeout: new(30 * time.Second)}`

2. 运行 go fix 的命令是什么？

   **答案**: `go fix ./...`
