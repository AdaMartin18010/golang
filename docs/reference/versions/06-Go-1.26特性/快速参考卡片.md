## 📝 语言特性

### new() 表达式

```go
// 语法
new(Expression)  // 返回 *T

// 示例
ptr := new(int(42))           // *int
ptr := new(MyStruct{Field: 1}) // *MyStruct
ptr := new(calculate())        // 计算值指针

// 使用场景
struct {
    Age  *int
    Name *string
}{
    Age:  new(calculateAge()),  // 简洁！
    Name: new(getName()),
}
```

| 传统写法 | Go 1.26 |
|----------|---------|
| `x := 42; p := &x` | `p := new(int(42))` |
| `tmp := f(); p := &tmp` | `p := new(f())` |

### 递归泛型约束

```go
// 语法
type C[T C[T]] interface { ... }

// 示例: 树节点
type TreeNode[T TreeNode[T]] interface {
    Value() int
    Children() []T
}

// 使用
func Traverse[T TreeNode[T]](root T) {
    for _, child := range root.Children() {
        Traverse(child)  // 递归调用
    }
}

// 实现
type BinaryNode struct { ... }
func (b *BinaryNode) Children() []*BinaryNode { ... }
```

**常见模式**:

- `Tree[T Tree[T]]` - 树结构
- `Ordered[T Ordered[T]]` - 可比较
- `Graph[N Node[N]]` - 图节点

## 📦 标准库

### crypto/hpke (混合加密)

```go
import "crypto/hpke"

// 1. 选择套件
kem, _ := hpke.GetKEM(hpke.KEM_MLKEM768)    // 后量子
kdf, _ := hpke.GetKDF(hpke.KDF_HKDF_SHA384)
aead, _ := hpke.GetAEAD(hpke.AEAD_AES256GCM)

suite, _ := hpke.NewSuite(kem, kdf, aead)

// 2. 接收方
sk, _ := kem.GenerateKeyPair()
pk := sk.PublicKey()

// 3. 发送方加密
sender := suite.NewSender(pk, nil)
enc, ctx, _ := sender.SetupBase()
ciphertext, _ := ctx.Seal(plaintext, nil)

// 4. 接收方解密
recipient := suite.NewRecipient(sk, nil)
ctx2, _ := recipient.SetupBase(enc)
plaintext, _ := ctx2.Open(ciphertext, nil)
```

**模式**: Base, PSK, Auth, AuthPSK

### errors.AsType (泛型断言)

```go
// 旧方式
var myErr *MyError
if errors.As(err, &myErr) { ... }

// Go 1.26 新方式
if myErr, ok := errors.AsType[*MyError](err); ok {
    // 使用 myErr
}
```

## 🛠️ 工具链

### go fix 现代化

```bash
# 应用所有 modernizers
go fix ./...

# 预览变化
go fix -n ./...

# 特定修复
go fix -fix=slicescontains ./...
```

**内置 Modernizers**:

- `slicescontains` - 循环 → `slices.Contains`
- `slicessort` - `sort.Slice` → `slices.Sort`
- `mapsclone` - 手动复制 → `maps.Clone`
- `minmax` - if-else → 内置 `min`/`max`
- `asany` - `errors.As` → `errors.AsType`

## 🎯 使用决策

### 何时使用 new(expr)？

```text
需要指针？
  ├─ 初始值需要计算？→ 使用 new(calculate())
  ├─ 可选字段？→ 使用 new(value)
  └─ 简单值？→ &T(v) 或 new(T(v)) 任选
```

### 何时使用递归泛型？

```text
需要通用树/图算法？
  ├─ 是 → 使用递归泛型
  └─ 否 → 使用普通泛型或具体类型
```
