## 2. 概念定义

### Dockerfile多阶段构建

**定义**: 优化镜像大小

**示例**:
```dockerfile
# 构建阶段
FROM golang:1.25.3 AS builder
WORKDIR /app
COPY . .
RUN go build -o main .

# 运行阶段
FROM alpine:latest
COPY --from=builder /app/main /main
CMD ["/main"]
```

---

### Kubernetes Deployment

**定义**: 声明式应用部署

**示例**:
```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: myapp
spec:
  replicas: 3
  template:
    spec:
      containers:
      - name: app
        image: myapp:latest
```
