## 核心概念

### 示例项目（Example Project）

**形式化定义**:

```go
type ExampleProject struct {
    Category    Category
    Difficulty  int
    CodeLines   int
    RunTime     time.Duration
    Description string
}
```

**特征**:

- 代码精简: 10-500行
- 单一概念: 专注演示
- 可运行: 开箱即用
- 注释充分: 易于理解

---

### 教程项目（Tutorial Project）

**形式化定义**:

```go
type TutorialProject struct {
    Level       Level
    Chapters    []Chapter
    Duration    time.Duration
    Goals       []Goal
    Projects    []Project
}
```

**特征**:

- 系统性: 完整知识体系
- 渐进式: 循序渐进
- 实战性: 项目驱动
- 评估: 学习检验
