## 1. 核心概念定义

### 1.1 示例项目（Example Project）

**形式化定义**:
```go
type Example struct {
    Category    ExampleCategory  // 示例分类
    Difficulty  DifficultyLevel  // 难度等级
    CodeLines   int              // 代码行数
    RunTime     time.Duration    // 运行时间
    Tags        []string         // 技术标签
    Description string           // 描述说明
}

type ExampleCategory int
const (
    BasicExample ExampleCategory = iota // 基础示例
    ConcurrencyExample                   // 并发示例
    WebExample                           // Web示例
    DatabaseExample                      // 数据库示例
    MicroserviceExample                  // 微服务示例
    FullProjectExample                   // 完整项目
)
```

**概念内涵**:
- 可独立运行的完整代码
- 演示特定技术或模式
- 包含必要的注释和说明
- 遵循Go最佳实践

**概念外延**:
- 从Hello World到完整项目
- 从10行到5000+行代码
- 从基础语法到企业架构
- 覆盖Go生态所有核心领域

### 1.2 学习路径（Learning Path）

**形式化定义**:
```go
type LearningPath struct {
    Level      SkillLevel        // 技能等级
    Examples   []Example         // 示例列表
    Duration   time.Duration     // 预计时长
    Goals      []LearningGoal    // 学习目标
    NextPath   *LearningPath     // 下一阶段
}

type SkillLevel int
const (
    Beginner SkillLevel = iota // 初级
    Intermediate                // 中级
    Advanced                    // 高级
    Expert                      // 专家
)
```

**路径设计原则**:
1. **循序渐进**: 从简到繁
2. **实战导向**: 贴近实际开发
3. **技能树**: 清晰的依赖关系
4. **里程碑**: 明确的阶段目标

**Go实践示例**:
```go
// 初级学习路径
beginnerPath := &LearningPath{
    Level: Beginner,
    Examples: []Example{
        {Name: "Hello World", Difficulty: 1},
        {Name: "Variables", Difficulty: 1},
        {Name: "Control Flow", Difficulty: 1},
    },
    Duration: 30 * 24 * time.Hour, // 1个月
    Goals: []LearningGoal{
        "掌握Go基础语法",
        "理解Go程序结构",
        "能写简单程序",
    },
}
```

## 2. 示例分类体系

### 2.1 基础示例（Basic Examples）

**定义**: 演示Go语言核心语法和标准库的简单示例

**特征**:
- 代码行数: 10-150行
- 无外部依赖或仅依赖标准库
- 单一概念聚焦
- 运行时间: < 1秒

**典型示例**:
```go
// Hello World - 最简单的Go程序
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

// 学习重点: 包声明、导入、main函数
```

---

### 2.2 并发示例（Concurrency Examples）

**定义**: 演示Go并发编程模型的示例

**核心概念**:
- Goroutine: 轻量级线程
- Channel: 通信机制
- Select: 多路复用
- 同步原语: Mutex, WaitGroup等

**典型模式**:
```go
// Worker Pool模式
type WorkerPool struct {
    numWorkers int
    jobs       chan Job
    results    chan Result
}

func (p *WorkerPool) Start() {
    for i := 0; i < p.numWorkers; i++ {
        go p.worker(i)
    }
}

func (p *WorkerPool) worker(id int) {
    for job := range p.jobs {
        result := job.Process()
        p.results <- result
    }
}

// 学习重点: 并发模式、资源管理、错误处理
```

### 2.3 Web示例（Web Examples）

**定义**: 演示Web开发技术的示例

**分类**:
1. **HTTP服务器**: net/http基础
2. **Web框架**: Gin/Echo/Fiber
3. **RESTful API**: CRUD操作
4. **实时通信**: WebSocket
5. **RPC**: gRPC服务
6. **GraphQL**: 查询API

**RESTful API示例**:
```go
// 使用Gin构建RESTful API
type UserController struct {
    service *UserService
}

func (c *UserController) Routes(r *gin.Engine) {
    users := r.Group("/api/users")
    {
        users.GET("", c.List)        // 列表
        users.GET("/:id", c.Get)     // 详情
        users.POST("", c.Create)     // 创建
        users.PUT("/:id", c.Update)  // 更新
        users.DELETE("/:id", c.Delete) // 删除
    }
}

// 学习重点: 路由设计、中间件、参数验证、错误处理
```

### 2.4 数据库示例（Database Examples）

**定义**: 演示数据库操作的示例

**覆盖技术**:
- SQL数据库: MySQL, PostgreSQL
- NoSQL: Redis, MongoDB
- ORM框架: GORM, ent
- 连接池管理
- 事务处理

**GORM示例**:
```go
// 用户模型
type User struct {
    gorm.Model
    Name     string `gorm:"size:100;not null"`
    Email    string `gorm:"size:100;uniqueIndex"`
    Age      int    `gorm:"check:age > 0"`
}

// CRUD操作
func CreateUser(db *gorm.DB, user *User) error {
    return db.Create(user).Error
}

func GetUser(db *gorm.DB, id uint) (*User, error) {
    var user User
    err := db.First(&user, id).Error
    return &user, err
}

// 学习重点: 模型定义、关系映射、查询优化
```

### 2.5 微服务示例（Microservice Examples）

**定义**: 演示微服务架构和相关技术的示例

**核心技术**:
- 服务通信: gRPC, REST
- 服务发现: Consul, etcd
- 配置中心: Consul, Apollo
- 链路追踪: Jaeger, Zipkin
- 熔断降级: Hystrix-Go
- 网关: 自研/Kong

**gRPC微服务示例**:
```go
// 定义服务
service UserService {
    rpc GetUser(GetUserRequest) returns (User);
    rpc ListUsers(ListUsersRequest) returns (stream User);
}

// 服务实现
type userServiceImpl struct {
    pb.UnimplementedUserServiceServer
    repo *UserRepository
}

func (s *userServiceImpl) GetUser(
    ctx context.Context,
    req *pb.GetUserRequest,
) (*pb.User, error) {
    user, err := s.repo.GetByID(ctx, req.Id)
    if err != nil {
        return nil, status.Error(codes.NotFound, "user not found")
    }
    return toProtoUser(user), nil
}

// 学习重点: 服务定义、错误处理、拦截器、负载均衡
```

**定义**: 具有完整功能的可上线项目

## 3. 学习路径设计

### 3.1 技能树模型

```text
Go技能树
│
├── 基础层 (必修)
│   ├── 语法基础 ✅
│   ├── 标准库 ✅
│   └── 工具链 ✅
│
├── 核心层 (必修)
│   ├── 并发编程 ✅
│   ├── Web开发 ✅
│   └── 数据库 ✅
│
├── 进阶层 (选修)
│   ├── 微服务 🎯
│   ├── 云原生 🎯

│
└── 专家层 (进阶)
    ├── 架构设计 🚀
```

- 完成20个基础示例

**第2阶段: 进阶（3-6月）**
- 完成30个中级示例
- 掌握Web框架和数据库
- 能开发完整的Web应用

**第3阶段: 高级（7-12月）**
- 完成20个高级示例
- 掌握微服务架构
- 能设计分布式系统
