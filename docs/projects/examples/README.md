# Go示例项目

Go示例项目集合，包含Web应用、微服务、CLI工具等实战案例。

---

## 📚 示例分类

1. **Web应用**: 博客系统、API服务
2. **微服务**: 用户服务、订单服务
3. **CLI工具**: 文件处理、系统监控
4. **库项目**: 工具库、框架

---

## 🎯 学习路径

### 初级 (1-2周)
- Hello World → CLI工具 → HTTP服务

### 中级 (2-3周)
- RESTful API → 数据库集成 → Web框架

### 高级 (3-4周)
- 微服务 → gRPC → 分布式系统
