#### 新手入门（1-2小时）

```bash
# Step 1: 并发基础

go test -v

# Step 2: 查看示例程序
cd pipeline_example
go run main.go
```

```bash
# Step 1: AI-Agent架构
cd examples/advanced/ai-agent
go test -v ./...

# Step 2: 现代化特性
cd examples/modern-features/03-stdlib-enhancements

```

```bash
# Step 1: Go 1.25运行时
cd examples/go125/runtime

# Step 2: 性能优化
cd examples/modern-features/07-performance-optimization
go test -bench=. -benchmem
```
