## 二、Stanford CS242 对齐

### 2.1 CS242: Programming Languages

```
课程结构 (2024-2025):
├── 模块1: 函数式编程基础
│   ├── 本指南对应: 函数值、闭包、高阶函数
│   └── Go实现: func作为值、闭包捕获、函数组合
│

│   ├── 本指南对应: 第3章 类型系统深度解析

│   ├── 泛型: 参数多态性、约束求解
│   └── 类型推断: Hindley-Milner扩展
│
├── 模块3: 面向对象
│   ├── 本指南对应: 接口、嵌入类型、方法集
│   └── Go特色: 隐式接口、组合优于继承
│
├── 模块4: 并发
│   ├── 本指南对应: 第5章 CSP形式化模型
│   ├── Actor模型对比CSP
│   └── 实际案例: Go运行时实现
│
└── 模块5: 形式化语义
    ├── 本指南对应: 第21章 形式化推理系统
    ├── 操作语义: 小步/大步语义
    └── 类型安全: 保持性、进展性证明

### 2.2 CS242作业映射

```
原始作业 (OCaml) → Go实现:

作业1: 表达式求值器

- Go: 第3章类型系统 + 接口实现

作业2: 类型推断
- OCaml: Hindley-Milner算法
- Go: 第4章泛型类型约束求解

作业3: 并发模拟
- OCaml: 协程库实现

项目: 语言实现
- OCaml: 完整解释器
- Go: 基于LLVM的JIT(第23章)

## 五、跨课程综合主题

### 5.1 类型系统深度

```
三校类型理论综合:

基础理论 (15-312):
├─ 简单类型λ演算
├─ 类型判断规则
└─ 归一化定理

扩展理论 (CS242):
├─ 递归类型
├─ 多态类型
└─ 子类型

应用理论 (6.822):
├─ 依赖类型 (轻量)
├─  refinement types
└─ 类型驱动开发

Go映射:
├─ 接口 = 存在类型
├─ 泛型 = System F_ω
└─ 方法 = 隐式参数传递
```

### 5.2 并发理论统一

```
并发模型比较:

            │   CSP    │   Actor    │   Threads+Locks
────────────┼──────────┼────────────┼─────────────────
同步方式     │ 同步通信  │ 异步消息   │ 共享内存
状态管理     │ 隔离      │ 封装      │ 显式同步
组合性      │ 良好      │ 良好       │ 复杂
形式化基础   │ 坚实      │ 相对弱     │ 复杂
Go实现      │ Channel   │ 无原生     │ sync包

Hoare CSP → Go映射的严格性:
├─ CSP进程 = Go goroutine (控制流)
├─ CSP事件 = Go channel操作 (同步点)
├─ CSP迹语义 = Go执行历史
└─ CSP精化 = Go行为包含
```

## 六、学习路径建议

### 6.1 本科生路径

```
Year 1: 基础建立
├── 完成: 01-04章 (语言基础)
├── 补充: CS61A (SICP) 或等效课程
└── 目标: 理解函数式思维

Year 2: 类型与并发
├── 完成: 03-10章 (类型+并发)
├── 补充: CS242 前半部分
└── 目标: 掌握CSP模型

Year 3: 系统与分布
├── 完成: 11-13章 (设计+分布式)
├── 补充: 6.824 (分布式系统)
└── 目标: 构建完整系统

Year 4: 形式化方法
├── 完成: 05,21章 (形式化)
├── 补充: 6.822 或 15-312
└── 目标: 严格验证能力
```

### 6.2 研究生路径

```
阶段1: 理论深化 (4周)
├── 精读: 05,21章
├── 课程: 15-312 前半
└── 实践: 实现小型类型检查器

阶段2: 并发验证 (4周)
├── 精读: 06,08,09章
├── 课程: 6.822 并发部分
└── 实践: 验证并发算法正确性

阶段3: 系统构建 (4周)
├── 精读: 12,13章
├── 课程: CS242 项目
└── 实践: 构建形式化验证的Go系统

阶段4: 研究前沿 (持续)
├── 领域: 语言设计/程序验证/系统可靠性
└── 产出: 论文或开源贡献
```
