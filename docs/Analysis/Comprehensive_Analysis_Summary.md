// 通用微服务架构模式

```go

    EventStore     *EventStore

}

// 命令处理器
type CommandHandler interface {
    Handle(ctx context.Context, command Command) error
}

// 查询处理器

    Handle(ctx context.Context, query Query) (interface{}, error)
}
```

**关键特性**:

- 读写分离的模型设计
- 命令和查询的明确分离
- 事件溯源支持
- 读写模型的独立优化
