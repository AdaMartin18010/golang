### 配置文件

- **Air 配置**: `.air.toml`
- **Codecov 配置**: `codecov.yml`
- **OpenAPI 规范**: `api/openapi/openapi.yaml`

```bash

make run-dev           # 开发模式运行
make test-coverage     # 测试覆盖率
make generate-openapi  # 生成 OpenAPI 代码
make validate-api      # 验证 API 规范
```
