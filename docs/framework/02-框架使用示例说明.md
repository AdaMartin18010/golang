# 框架使用示例说明

> **版本**: v1.0
> **日期**: 2025-01-XX

---

## 📋 概述

框架提供了完整的使用示例，展示如何使用框架构建应用。这些示例代码位于 `examples/framework-usage/` 目录。

## ⚠️ 重要说明

**这些是示例代码，不是框架的一部分**：

- ✅ 展示框架的使用方式
- ✅ 展示 Clean Architecture 的应用
- ✅ 展示领域模型设计
- ❌ 不是框架的核心代码
- ❌ 不应在生产环境中直接使用

## 📁 示例结构

```text
examples/framework-usage/
├── user-domain/     # 用户领域示例
│   ├── domain/      # 领域层示例
│   ├── application/ # 应用层示例
│   └── interfaces/  # 接口层示例
└── README.md        # 示例说明
```

## 🎯 示例目的

1. **学习参考**: 展示如何使用框架
2. **最佳实践**: 展示 Clean Architecture 最佳实践
3. **快速开始**: 帮助用户快速上手

## 🚀 如何使用示例

### 1. 查看示例代码

```bash
# 查看用户领域示例
cat examples/framework-usage/user-domain/README.md
```

### 2. 参考示例实现

参考示例代码的实现方式，在自己的项目中：

1. 定义领域模型（通过 Ent Schema）
2. 实现应用服务
3. 实现接口层

### 3. 运行示例（如果提供）

```bash
# 运行示例（如果提供）
cd examples/framework-usage/user-domain
go run main.go
```
