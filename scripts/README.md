# Scripts 工具集

本目录包含项目维护和自动化工具。

---

## 📋 工具列表

### 1. 项目统计工具 (project_stats)

**功能**: 自动分析和统计项目信息

**使用方法**:

```bash
# 在项目根目录运行
cd scripts/project_stats
go run main.go

# 或指定目录
go run main.go /path/to/project
```

**输出内容**:

- 📊 总体统计 (文件数、行数、字数)
- 💻 代码统计 (示例数、测试数)
- 📚 文档分类
- 📁 文件类型分布
- 🎯 项目指标
- ✨ 质量评估

---

### 2. 变更日志生成器 (gen_changelog)

**功能**: 自动生成项目变更日志

**使用方法**:

```bash
cd scripts/gen_changelog
go mod tidy

# 生成变更日志
echo "Added PGO example" | VERSION=v2025.09-P1 go run main.go
```
