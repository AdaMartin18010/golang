## 💻 代码示例

### WaitGroup.Go() - 更简洁的并发

**Go 1.24**:

```go
var wg sync.WaitGroup
wg.Add(1)
go func() {
    defer wg.Done()
    // 工作
}()
wg.Wait()
```

**Go 1.23+**:

```go
var wg sync.WaitGroup
wg.Go(func() {
    // 工作
})
wg.Wait()
```

节省 **75% 代码**，减少错误！

### 容器感知调度 - 零配置优化

```go
// Go 1.23+ 自动检测容器 CPU 限制
fmt.Printf("GOMAXPROCS: %d\n", runtime.GOMAXPROCS(0))
// 在 Kubernetes 中自动适配 Pod 的 CPU 配额！
```
