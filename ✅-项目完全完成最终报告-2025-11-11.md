## 🚀 快速开始

### 本地开发

```bash
# 1. 安装依赖
go mod tidy

# 2. 启动 HTTP 服务器
make run
# 或
go run ./cmd/server

# 3. 启动 Temporal Worker（新终端）
make run-worker
# 或
go run ./cmd/temporal-worker
```

### Docker 部署

```bash
# 启动所有服务
docker-compose -f deployments/docker/docker-compose.yml up -d

# 查看服务状态
docker-compose -f deployments/docker/docker-compose.yml ps

# 查看日志
docker-compose -f deployments/docker/docker-compose.yml logs -f app
docker-compose -f deployments/docker/docker-compose.yml logs -f temporal-worker

# 停止服务
docker-compose -f deployments/docker/docker-compose.yml down
```

### 测试

```bash
# 运行所有测试
make test

# 运行单元测试
go test ./test/unit/...

# 测试覆盖率
make test-coverage
```
