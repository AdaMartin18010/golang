# 04-数据库编程

> Go语言数据库编程完全指南

---

## 📚 模块概览

本模块全面介绍Go语言中的数据库编程,包括SQL数据库、NoSQL数据库、ORM框架使用和数据库优化等内容。

#### MySQL

- 连接与配置
- CRUD操作

- 连接池管理
- 性能优化

#### PostgreSQL

- 连接与配置
- 高级特性
- JSON支持
- 全文搜索
- 性能调优

#### SQLite

- 嵌入式数据库
- 应用场景
- 最佳实践

#### Redis

- 基础操作

- 发布订阅

- 性能优化

#### MongoDB

- 文档操作
- 聚合查询
- 索引优化
- 副本集
- 分片集群

#### Elasticsearch

- 全文搜索
- 聚合分析
- 性能优化
