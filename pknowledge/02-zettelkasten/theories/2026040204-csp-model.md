## 定义

由 Tony Hoare (1978) 提出的并发计算形式化语言。

## 核心概念

### 进程 (Process)

- 独立的计算单元
- 通过通信交互

### 通道 (Channel)

- 进程间通信媒介
- 同步/异步语义

### 通信原语

```
send(c, v)    - 向通道 c 发送值 v
receive(c)    - 从通道 c 接收值
```

## Go 的 CSP 实现

```go
// Goroutine = Process
// Channel = Channel

go func() {      // 启动新进程
    ch <- value  // 发送
}()

value := <-ch    // 接收
```

## 形式化语义

### 语法 (简化)

```
P ::= 0 | a.P | P + P | P | P | (νa)P
```

### Go 映射

```
0       → 终止
c!v.P   → ch <- v; P
c?x.P   → x := <-ch; P
P|Q     → go P(); Q()
(νc)P   → c := make(chan T); P
```

## Happens-Before 关系

### 公理

1. `go stmt` happens-before goroutine 执行
2. `send(c, v)` happens-before `receive(c) → v`
3. `close(c)` happens-before `receive(c) → zero`
4. `unlock(m)` happens-before `lock(m)`

### 可观察性

```
read r observes write w iff:
  ¬(r <hb w) ∧ ¬∃w': w <hb w' <hb r
```

## 与 π-演算关系

Go channels 可编码 π-calculus：

- 通道作为 first-class 值
- 支持 mobility (通道传递)
- 动态拓扑变化

## 证明技术

### Trace Semantics

进程行为表示为事件序列。

### Bisimulation

证明两个进程行为等价。
