## 💡 FAQ特色功能

### 1. 对比示例

**错误 vs 正确**:
```markdown
// ❌ 错误：永远运行
go func() {
    for {
        doWork()
    }
}()

// ✅ 正确：可以退出
go func() {
    for {
        select {
        case <-ctx.Done():
            return
        default:
            doWork()
        }
    }
}()
```
