## 日志分析

```go
// 解析日志
scanner := bufio.NewScanner(file)
for scanner.Scan() {
    line := scanner.Text()
    // 解析 JSON 日志
    var log LogEntry
    json.Unmarshal([]byte(line), &log)
}
```
