# 代码审查 (Code Review)

> **分类**: 工程与云原生

---

## 审查清单

### 功能性

- [ ] 代码是否实现了需求？
- [ ] 边界条件是否处理？
- [ ] 错误是否正确处理？

### 可读性

- [ ] 命名是否清晰？
- [ ] 函数是否简洁？
- [ ] 注释是否必要且准确？

### 性能

- [ ] 是否有明显性能问题？
- [ ] 资源是否正确释放？

---

## Go 特定检查点

### 错误处理

```go
// ✅ 好的: 处理或注释
if err := file.Close(); err != nil {
    log.Printf("close file: %v", err)
}
```

### 并发安全

```go
// ✅ 好的: 同步
var mu sync.Mutex
go func() {
    mu.Lock()
    defer mu.Unlock()
    counter++
}()
```

❌ "这明显是错的"
