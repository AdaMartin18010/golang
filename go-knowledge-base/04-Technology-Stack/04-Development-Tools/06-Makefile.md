## 基本结构

```makefile
.PHONY: build test clean

build:
    go build -o bin/app .

test:
    go test -v ./...

clean:
    rm -rf bin/
```

## 变量

```makefile
BINARY=app
VERSION=1.0.0
LDFLAGS=-ldflags "-X main.version=$(VERSION)"

build:
    go build $(LDFLAGS) -o bin/$(BINARY) .
```
