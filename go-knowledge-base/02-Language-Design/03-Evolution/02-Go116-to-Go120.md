# Go 1.16 - 1.20 演进

> **分类**: 语言设计

---

## Go 1.16 (2021)

**特性**:

- **embed 包**: 文件嵌入
- **io/fs 包**: 文件系统抽象
- **go mod vendor**: 默认行为改进

```go
// embed 示例
import _ "embed"

//go:embed schema.sql
var schema string

//go:embed static/*
var static embed.FS
```

---

## Go 1.18 (2022) ⭐️ 里程碑

**泛型正式发布**

```go
// 泛型示例
func Map[T, U any](s []T, f func(T) U) []U {
    r := make([]U, len(s))
    for i, v := range s {
        r[i] = f(v)
    }
    return r
}
```

**其他**:

- 工作区模式 (实验性)
- 模糊测试 (fuzzing)

---

## Go 1.20 (2023)

**特性**:

- **PGO (Profile-Guided Optimization)**: 性能提升 2-4%
- 切片到数组转换
- 上下文取消原因

```go
// PGO 使用
// go build -pgo=cpu.pprof

// 切片转数组
array := (*[4]byte)(slice)

// 取消原因
ctx, cancel := context.WithCancelCause(ctx)
cancel(errors.New("custom reason"))
```

## 主要改进

| 版本 | 特性 | 影响 |
|------|------|------|
| 1.16 | embed | 单二进制部署 |
| 1.18 | 泛型 | 类型安全抽象 |
| 1.20 | PGO | 自动性能优化 |
