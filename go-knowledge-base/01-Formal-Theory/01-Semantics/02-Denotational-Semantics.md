# 指称语义 (Denotational Semantics)

> **分类**: 形式理论

---

## 核心思想

将程序**映射**到数学对象（通常是函数）。

```
〚e〛 ∈ Environment → Value
```

---

## 基本定义

### 环境

```
Env = Variable → Value
```

### 表达式语义

```
〚n〛(ρ) = n                           (常量)

〚x〛(ρ) = ρ(x)                        (变量)

〚e₁ + e₂〛(ρ) = 〚e₁〛(ρ) + 〚e₂〛(ρ)    (加法)
```

---

## Go 示例

### 结构体

```
〚t_S{e₁,...,eₙ}〛(ρ) =
  (t_S, {f₁: 〚e₁〛(ρ), ..., fₙ: 〚eₙ〛(ρ)})
```

### 方法调用

```
〚e.m()〛(ρ) = body(m, type(〚e〛(ρ)))(〚e〛(ρ))
```

---

## 与操作语义对比

| 指称语义 | 操作语义 |
|----------|----------|
| 是什么 | 怎么做 |
| 数学函数 | 执行步骤 |
| 适合证明等价 | 适合实现 |
